# 数据集介绍
  ## 简介
  该数据集仅用于研究目的。引文数据从 DBLP、ACM、MAG（Microsoft Academic Graph）和其他来源中提取。第一版包含 629,814 篇论文和 632,752 次引用。每篇论文都与摘要、作者、年份、地点和标题相关联。
该数据集可用于网络和边信息的聚类、研究引文网络中的影响力、寻找最有影响力的论文、主题建模分析等。
